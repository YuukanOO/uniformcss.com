## Colors

Customizing the color scheme to match your brand is an important aspect of Uniform CSS. Colors in Uniform are flexibile and semantically organized to fit standard brand design systems. Uniform also provides powerful utility-based manipulation features that provides both saturation and lightness control.

## Advanced Color Manipulation

All colors in Uniform CSS are defined using HSL. This opens up the opportunity of adjusting saturation and lightness using utility properties. By doing so, colors can vary in both vibrancy and shade without drastically affecting the final CSS output size.

<div class="bg-silver-200 p-20 h-200px radius-md">
  <div class="grid grid-cols-4 leading-6 gap-18">
    <div class="ratio-square radius-sm bg-red"></div>
    <div class="ratio-square radius-sm bg-red bg-brighten-500"></div>
    <div class="ratio-square radius-sm bg-red bg-darken-200"></div>
    <div class="ratio-square radius-sm bg-red bg-lighten-800"></div>
  </div>
</div>

```html
<div class="bg-red ..."></div>
<div class="bg-red bg-brighten-500 ..."></div>
<div class="bg-red bg-darken-200 ..."></div>
<div class="bg-red bg-lighten-800 ..."></div>
```

### How Color Manipulation Works

The advanced color manipulation feature is only available to `background-color`, `color`, and `border-color` properties. For each type of manipulation the utility shorthand of the property you're trying to affect must be included.

<div class="bg-white border-1 border-silver-300 p-20 h-200px radius-md">
  <div class="grid grid-cols-1 leading-6 gap-18">
    <div class="flex align-items-center justify-content-center h-48 align-center radius-sm text-blue text-lighten-300">Light blue text</div>
    <div class="flex align-items-center justify-content-center h-48 align-center radius-sm bg-blue bg-darken-400 text-white">Dark blue background</div>
    <div class="flex align-items-center justify-content-center h-48 align-center radius-sm border-4 border-red border-lighten-800 border-brighten-500">Light and bright red border</div>
  </div>
</div>

```html
<div class="text-blue text-lighten-300 ..."></div>
<div class="bg-blue bg-darken-400 ..."></div>
<div class="border-red border-lighten-800 border-brighten-500 ..."></div>
```
