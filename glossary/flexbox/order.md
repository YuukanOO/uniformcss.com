## Basic Usage

The `order` utilities control layout order for Flexbox and CSS Grid chidlren.

| Class Name    | Property       |
| ------------- | -------------- |
| `order-first` | `order: -9999` |
| `order-last`  | `order: 9999`  |
| `order-1`     | `order: 1`     |
| `order-2`     | `order: 2`     |
| `order-3`     | `order: 3`     |
| `order-4`     | `order: 4`     |
| `order-5`     | `order: 5`     |
| `order-6`     | `order: 6`     |
| `order-7`     | `order: 7`     |
| `order-8`     | `order: 8`     |
| `order-9`     | `order: 9`     |
| `order-10`    | `order: 10`    |
| `order-11`    | `order: 11`    |
| `order-12`    | `order: 12`    |
