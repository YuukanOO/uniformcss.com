## Basic Usage

The `flex` utilities set `flex-grow`, `flex-shrink`, and `flex-basis`.

| Class Name     | Property         |
| -------------- | ---------------- |
| `flex-initial` | `flex: 0 1 auto` |
| `flex-1`       | `flex: 1 1 0%`   |
| `flex-auto`    | `flex: 1 1 auto` |
| `flex-none`    | `flex: none`     |

---

## Default Settings

The following default responsive and pseudo settings.

| Setting      | Default Setting |
| ------------ | --------------- |
| `responsive` | `true`          |
| `pseudos`    | `none`          |
