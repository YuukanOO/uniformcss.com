## Basic Usage

The `clear` utilities clear negative space around floated elements.

| Class Name    | Property       |
| ------------- | -------------- |
| `clear-left`  | `clear: left`  |
| `clear-right` | `clear: right` |
| `clear-both`  | `clear: both`  |
