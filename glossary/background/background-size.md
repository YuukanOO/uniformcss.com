## Basic Usage

The `background-size` utilities set the background image size.

| Class Name   | Property                   |
| ------------ | -------------------------- |
| `bg-auto`    | `background-size: auto`    |
| `bg-cover`   | `background-size: cover`   |
| `bg-contain` | `background-size: contain` |
