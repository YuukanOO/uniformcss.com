## Basic Usage

The `font-style` utilities set font style.

| Class Name | Property             |
| ---------- | -------------------- |
| `italic`   | `font-style: italic` |

---

## Default Settings

The following default responsive and pseudo settings.

| Setting      | Default Setting |
| ------------ | --------------- |
| `responsive` | `none`          |
| `pseudos`    | `none`          |
