## Basic Usage

The `text-overflow` utilities set text overflow behavior.

| Class Name | Property                  |
| ---------- | ------------------------- |
| `ellipsis` | `text-overflow: ellipsis` |

---

## Default Settings

The following default responsive and pseudo settings.

| Setting      | Default Setting |
| ------------ | --------------- |
| `responsive` | `none`          |
| `pseudos`    | `none`          |
