## Basic Usage

The `color-overflow` utilities set text overflow behavior.

| Class Name | Property                  |
| ---------- | ------------------------- |
| `ellipsis` | `color-overflow: ellipsis` |

{.table}

---

## Default Settings

The following default responsive and pseudo settings.

| Setting      | Default Setting |
| ------------ | --------------- |
| `responsive` | `none`          |
| `pseudos`    | `none`          |
