## Basic Usage

The `list-style` utilities set the decoration for list item markers..

| Class Name     | Property                   |
| -------------- | -------------------------- |
| `list-none`    | `list-style-type: none`    |
| `list-disc`    | `list-style-type: disc`    |
| `list-square`  | `list-style-type: square`  |
| `list-decimal` | `list-style-type: decimal` |
