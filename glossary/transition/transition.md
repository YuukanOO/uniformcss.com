## Basic Usage

The `transition` utilities enable composition using the [Transition API].

| Class Name   | Property                                                                                                   |
| ------------ | ---------------------------------------------------------------------------------------------------------- |
| `transition` | `transition-property: background-color, border-color, color, fill, stroke, opacity, box-shadow, transform` |
