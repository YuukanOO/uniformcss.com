## Basic Usage

The `resize` utilities control along which axes an element is resizable.

| Class Name         | Property             |
| ------------------ | -------------------- |
| `resize-none       | `resize: none`       |
| `resize-both       | `resize: both`       |
| `resize-horizontal | `resize: horizontal` |
| `resize-vertical   | `resize: vertical`   |
