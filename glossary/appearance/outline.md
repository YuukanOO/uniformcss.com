## Basic Usage

The `outline` utilities control outline-rendering. Note that outlines, unlike borders, do not reflow layout.

| Class Name  | Property     |
| ----------- | ------------ |
| `outline-0` | `outline: 0` |

---

## Default Settings

The following default responsive and pseudo settings.

| Setting      | Default Setting |
| ------------ | --------------- |
| `responsive` | `none`          |
| `pseudos`    | `none`          |
