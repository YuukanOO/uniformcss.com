## Basic Usage

The `appearance` utilities control whether to style inputs using browser defaults.

| Class Name        | Property           |
| ----------------- | ------------------ |
| `appearance-none` | `appearance: none` |

{.table}

---

## Default Settings

The following default responsive and pseudo settings.

| Setting      | Default Setting |
| ------------ | --------------- |
| `responsive` | `none`          |
| `pseudos`    | `none`          |
