## Basic Usage

The `appearance` utility controls whether to use browser styles for inputs.

| Class Name        | Property           |
| ----------------- | ------------------ |
| `appearance-none` | `appearance: none` |

---

## Default Settings

The following default responsive and pseudo settings.

| Setting      | Default Setting |
| ------------ | --------------- |
| `responsive` | `none`          |
| `pseudos`    | `none`          |
